Kennedy HTML/CSS Cheatsheet

HTML Cheatsheet

HTML Elements
Input Effect
a Creates a link to another page, include the link with href
audio Inserts an audio file into the HTML, takes src. Combine with controls / autoplay
HTML Attributes
Input Effect
src Allows an image to be embedded in an html. Takes a source (src) in the element.
id Sets a unique id to an HTML element, allowing it to be seperately edited in CSS.
div Creates a division in HTML.
em Emphasizes text (usually with italics).
head Immediate child of html, contains metadata.
img Embeds an image in html, utilizes src for source, and alt to display text if image does not render.
li Represents a single list item, follows ol, ul or menu element.
link Connects current page with an external file.
ol Represents the beginning of an organized list. Used with li.
span Used for grouping related text or elements in a single line.
strong Used to bold text.
table Represents the beginning of a table, used with tr, th, td
ul Represents the beginning of an unorganized list. Used with li.
video Embeds a video. Used with Source. Recommend using controls

CSS Cheatsheet

CSS Properties
Input Effect
background-color Sets the background color of an element.
background-image Sets an image as the background of an element. Utilizes url().
border Adjusts settings for a border around an element.
display Specifies how an HTML element should be displayed. Utilizes inline, block, inline-block.
float Floats an element. Top, Bottom, Left, Right.
font-family Sets the font of an element.
font-size Sets the size of text.
font-weight Specifies thickness of a font.
letter-spacing Sets the amount of space between letters.
line-height Sets distance between lines.
margin Adjusts the margin distance. Can be used with top, right, left or bottom.
opacity Adjusts the opacity of an element.
overflow Specifies what content should do if it overflows its element. Utilizes scroll, hidden, auto, clip, visible.
padding Adjusts the padding of an element. Can be used with top, right, bottom, left.
position Specifies the type of positioning for an element. Utilizes static, relative, absolute, fixed.
text-align Specifies the horizontal alignment of text. Utilizes left, right, center, or justify.
text-indent Specifies the indentation of the first line in a text-block.
z-index Sets the stack order of overlapping elements.
